# 1. 命名规则

## 1.1. 国家

国家一般以国家的缩写来表示

| Country | Description  |      |
| ------- | ------------ | ---- |
| cn      | China        |      |
| us      | United State |      |
| en      | English      |      |

## 1.2. 地区

地区一般以城市的缩写来表示

| Region | Description |      |
| ------ | ----------- | ---- |
| bj     | Beijing     |      |
| sh     | Shanghai    |      |

## 1.3. 数据中心

数据中心

| Datacenter | Description       |      |
| ---------- | ----------------- | ---- |
| p          | Primary           |      |
| s          | Secondary         |      |
| d          | Disaster recovery |      |

## 1.4. 平台

平台用来表示用什么平台来承载了这个lun

| Platform | Description     |      |
| -------- | --------------- | ---- |
| v        | vSAN            |      |
| s        | SAN             |      |
| n        | NAS(nfs/smb)    |      |
| i        | iSCSI           |      |
| o        | Object Storage  |      |
| e        | ESXi local disk |      |

## 1.5.环境

环境用来表示环境的信息

| Evironment | Description |      |
| ---------- | ----------- | ---- |
| p          | Production  |      |
| u          | UAT         |      |
| t          | Test        |      |
| d          | Development |      |
